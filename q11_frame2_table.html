Table

A table in HTML consists of table cells inside rows and columns.

Table Rows

Each table row starts with a <tr> and end with a </tr> tag.

Table Data

Each table cell is defined by a <td> and a </td> tag. Everything between <td> and </td> are the content of the table cell.

Table Head

The <th> tag defines a header cell in an HTML table.
The text in elements are bold.

Caption

The <caption> tag defines a table caption.
The <caption> tag must be inserted immediately after the <table>> tag.

Table Attributes

The attributes of table tag are:
Align, bgcolor, border, cellpadding, cellspacing, rules, width, height, bordercolor.
