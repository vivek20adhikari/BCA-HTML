Frame Tag

HTML <frame> tag are used to divide the web browser window into multiple sections where each section can be loaded separately.
Attribute used in frame tag is src and name.

Frameset Tag

To use frames on a page we use <frameset> tag instead of <body> tag. The <frameset> tag defines, how to divide the window into frames.
Attribute- The rowsattribute of <frameset> tag defines horizontal frames and cols attribute defines vertical frames.
Each frame is indicated by <frame> tag and it defines which HTML document shall open into the frame.
