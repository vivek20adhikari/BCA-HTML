LISTS

There are Five types of List in Html
1.Order List
Order List starts with the <ol> tag. It is Container tag.Each list item starts with the <li> tag.
Attribute-The there are 5 types of attributes in <ol>tag.
Numeric Number (1, 2, 3), Capital Roman Number (I,II,III), Small Romal Number (i,ii,iii), Capital Alphabet (A,B,C), Small Alphabet (a,b,c)
There is another attribute called start. It help types that where they will start.

2.Unorder List
Order List starts with the <ul> tag. It is Container tag. Each list item starts with the <li> tag.
Attribute-The there are 4 types of attributes in <ul>tag.
disc, circle, square, none

3.Definition List
The <dl> tag defines the description list, the <dt> tag defines the term (name), and the <dd> tag describes each term.It is Container tag.
